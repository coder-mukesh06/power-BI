# 📊 Adidas Sales Dashboard (Power BI)

## 🔍 Project Overview

This project analyzes Adidas sales data to identify key business insights such as revenue trends, profit performance, and regional sales distribution.

## 🎯 Objectives

* Track total sales and profit
* Analyze region-wise performance
* Identify top-performing products
* Understand sales trends over time

## 📈 Key KPIs

* Total Sales
* Total Profit
* Profit Margin
* Top Region by Sales
* Best Selling Product

## 🛠️ Tools Used

* Power BI
* DAX
* Data Cleaning
